How to Use MARINA

Getting Started

MARINA uses deep learning to predict molecular structures from spectral data. You can input various types of spectroscopic data to get structure predictions.

Workflow:

  1. Enter Spectral Data: Input HSQC NMR, H NMR, C NMR, Mass Spec, or Molecular Weight data
  2. Review Input Summary: Check what data you've entered before predicting
  3. Predict Structure: Click "Predict Structure" to get molecular structure predictions
  4. Analyze Results: Review the predicted structures and their similarity scores

Spectral Data Input

Keyboard Shortcuts:

  • Ctrl+V (or Cmd+V): Paste data from clipboard
  • Ctrl+C (or Cmd+C): Copy selected data
  • Arrow keys: Navigate between cells
  • Tab: Move to next cell
  • Enter: Move to next row

HSQC NMR:

  • Enter 1H chemical shift (ppm), 13C chemical shift (ppm), and intensity values
  • Each row represents one HSQC correlation
  • Use the "Swap H↔C" button if your data has 13C and 1H in different columns

H NMR:

  • Enter 1H chemical shift values in ppm
  • One value per row

C NMR:

  • Enter 13C chemical shift values in ppm
  • One value per row

Mass Spectrometry:

  • Enter m/z and intensity values
  • Each row represents one mass peak

Molecular Weight:

  • Enter the molecular weight in g/mol
  • Optional but can improve prediction accuracy

SMILES Search

The SMILES search allows you to find similar molecules in the database using a SMILES string.

How to Use:

  • Enter a valid SMILES string in the search box
  • Click "Search" to find similar molecules
  • Results show molecular structures, SMILES, and similarity scores

Example SMILES:

  • CN1C=NC2=C1C(=O)N(C(=O)N2C)C - Caffeine
  • CC(=O)OC1=CC=CC=C1C(=O)O - Aspirin
  • C([C@@H]1[C@H]([C@@H]([C@H]([C@H](O1)O)O)O)O)O - Glucose
  • COC1=C(CCC(C)(C)O)C(=O)NC2=C(O)C=CC=C12 - Ruticarponine A

Understanding Results

Similarity Scores:

  • Scores range from 0% to 100%
  • Higher scores indicate more similar molecular structures
  • Scores are based on molecular fingerprint similarity

Molecular Structures:

  • Each result shows the 2D molecular structure
  • Molecular formula and molecular weight are displayed
  • SMILES strings are provided for each structure

Tips for Better Results:

  • Provide as much spectral data as possible
  • Ensure chemical shift values are accurate
  • Include molecular weight when available
  • Try different combinations of spectral data
