# MARINA Website

## Installation

Just install pixi and npm

## Dev running (no docker)
Enter `backend/` and run in two windows
```
pixi run dev-marina
```
```
pixi run dev-spectre
```
Enter `frontend/` and run
```
npm run dev
```
(If first time running, run `npm i` first)

## Prod running (docker)
```
bash restart.sh
```
